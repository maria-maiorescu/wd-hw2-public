Junior Developer Project Showcase

Welcome

Welcome to my project showcase website. I am Maria Maiorescu, a first-year Computer Science student at UVT, and this website presents some of my programming work.

Here you can find information about my main projects, my interests, and a structured overview of this mini-project.

Featured Projects

Dive Log

Dive Log is a C++ console application created for managing and analyzing scuba diving data. It includes functions for displaying dives, editing entries, searching by location, and generating useful statistics.

Bakery Management Application

This project focuses on organizing bakery-related information and product data. It shows my ability to work on practical software ideas from different domains.

Projects in Progress

Sorting Algorithms Project

This project is written in C and focuses on comparing sorting algorithms and studying their performance. It also includes a research paper that will be written in LaTeX about the algorithms, their complexity, and their behavior on different datasets.

Parking Payment System

This is an interactive C project based on a parking payment system. The user selects the parking zone, enters the plate number, street, and number of hours spent, then receives the amount to pay. The project also includes a parking history feature.

In the future, the application will be extended with file reading and writing and will be improved using C++ for better structure and functionality.
