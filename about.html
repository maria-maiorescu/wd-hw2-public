About Me and My Projects

About Me

My name is Maria Maiorescu and I am a first-year Computer Science student at UVT. I am interested in programming, problem solving, and building projects that combine technical knowledge with real-world ideas.

This website presents some of my work as a junior developer. It focuses on projects that helped me practice structured thinking, data organization, and clear presentation.

Dive Log Project

Dive Log is one of my main projects and is based on a real personal interest: scuba diving. The application was developed in C++ and works as a console program for managing and analyzing dive records.

The project can import data from CSV files, display dive information, search dives by location, and generate statistics such as the longest dives, deepest dives, and air consumption analysis.

This project helped me improve my understanding of file processing, structured data, modular programming, and practical problem solving.

Bakery Management Application

The Bakery Management Application is another important project in my portfolio. It focuses on organizing bakery-related information and handling product data in a structured way.

This project shows that I can work on software ideas from different domains. It also helped me practice program structure, data handling, and presenting useful information clearly.

Projects in Progress

Sorting Algorithms Project

This project is written in C and focuses on comparing sorting algorithms and analyzing their performance. It will also include a research paper written in LaTeX about algorithm complexity and behavior.

Parking Payment System

This project is an interactive application written in C that simulates a parking payment system. The user selects the parking zone, enters the plate number, street, and number of hours, and receives the total cost.

The project also includes a parking history feature. In the future, it will be extended with file reading and writing and improved using C++.
