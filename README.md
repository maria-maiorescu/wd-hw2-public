# Portfolio Website - Web Design hw 2

A multi-page portfolio website built with plain HTML and CSS, made as a homework assignment for the Web Development course at UVT.

## Pages

- **Home** (`index.html`) — intro and featured projects overview
- **About** (`about.html`) — short bio and background
- **Data** (`data.html`) — projects listed in a table with type, technology, and status
- **Contact** (`contact.html`) — contact form
- **Overview** (`overview.html`) — site structure overview
- **Card** (`card.html`) — profile card with name, role, and university info
- **Chatbot** (`chatbot.html`) — simple keyword-based Q&A chatbot about scuba diving
- **Specificity** (`specificity.html`) — a CSS specificity demo/game

## Features

- Consistent header and navigation across all pages
- Skip-to-content link for keyboard accessibility
- `aria-current="page"` on nav links to show the active page
- Scuba diving chatbot powered by plain JavaScript — no libraries
- CSS specificity demo page showing how selectors override each other
- Figma sign-in page mockup included in the `figma/` folder
- No frameworks — just HTML, CSS, and a bit of JS
